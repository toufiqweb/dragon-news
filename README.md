
#  Dragon News

A modern, dynamic, and fully responsive news application built with **Next.js**. Dragon News delivers category-based news content with authentication, detailed news pages, and a clean user experience across all devices.

----------

## 🚀 Live Project

-   **Live Website:** https://dragon-news-ochre.vercel.app/category/01
    
-   **GitHub Repository:**https://github.com/toufiqweb/dragon-news
    

----------

## 📌 Project Overview

**Dragon News** is a dynamic news platform where users can browse news by categories, view detailed articles, and access protected sections after authentication.

The project focuses on a smooth user experience, responsive UI, secure authentication, and category-based dynamic news loading.

## ✨ Features

### 🏠 Homepage Layout

-   Custom banner section with current date display
    
-   Breaking news marquee section
    
-   Navigation bar with:
    
    -   Home
        
    -   About
        
    -   Career
        
    -   Login button
        
-   Organized homepage layout with:
    
    -   All News section
        
    -   Dragon News section
        
    -   Social links section
        

### 📰 Category Based News

-   Dynamic category-wise news loading
    
-   Categories include:
    
    -   Breaking News
        
    -   Regular News
        
    -   Sports
        
    -   Entertainment
        
-   API-based category filtering
    
-   Global and local loading states for better UX
    

### 🔐 Authentication System

-   User Registration
    
-   Email & Password Login
    
-   Social Authentication:
    
    -   Google Login
        
    -   GitHub Login
        
-   Authentication data stored using MongoDB
    

### 🔒 Protected Routes

-   News Details page restricted to logged-in users
    
-   Career section accessible only for authenticated users
    

### 📄 News Details Page

-   Dynamic route for individual news details
    
-   "Read More" button for full article access
    
-   "Read More News" button to return to previous category
    

### 📱 Responsive Design

-   Fully responsive across:
    
    -   Mobile devices
        
    -   Tablets
        
    -   Laptops
        
    -   Desktop screens
       

## 🛠️ Tech Stack

### Frontend

-   **Next.js**
    
-   **Tailwind CSS**
    
-   **DaisyUI**
    
-   **Lucide React Icons**
    

### Backend & Database

-   **MongoDB**
    

### Utilities

-   **date-fns** for date formatting and display

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/toufiqweb/dragon-news.git

```

### Navigate to project directory

```bash
cd dragon-news

```

### Install dependencies

```bash
npm install

```

### Run development server

```bash
npm run dev

```

Open:

```bash
http://localhost:3000

```
